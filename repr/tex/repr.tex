\hhh{当执行 git log --all --oneline 时}

\begin{itemize}
\item
  git log 用于显示commit信息
\item
  --all 选项会显示所有分支的提交历史，而不仅仅是当前分支。
\item
  --oneline 选项会使每个提交只显示一行，通常包括提交的哈希值和提交信息。
\end{itemize}
